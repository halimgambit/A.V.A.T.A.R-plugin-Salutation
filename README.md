# Salutation
Ce module reconnaît les formules de politesse et répond aux salutations de l'utilisateur

- This plugin is an add-on for the [A.V.A.T.A.R](https://avatar-home-automation.github.io/docs) framework. 

 ## 🎯 Usage
Commandes :
- bonjour, bonsoir, dit bonjour à pierre, dis bonsoir à pierre
- dis bonjour à tout le monde, dis bonsoir à mon professeur

## Multi-room

The `Salutation` plugin is fully multi-room.

## Multi-language

The `Salutation` plugin relies solely on the system's available languages.
